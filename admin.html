🎁 Gift Overlay Admin

セットの追加・編集・並び替え → JSON書き出し → リポジトリにpush

セット一覧

使い方:
1. 「＋ セット追加」で新規セット追加、または「📂 JSON読み込み」で既存data.jsonを編集再開
2. 各セットのサムネイル枠に画像ファイルをドラッグ&ドロップ(またはクリックで選択)。同名ファイルがダウンロードされ、画像パスが自動入力される
3. ギフト名・任意テキストを入力、必要に応じて↑↓ボタンで並び替え
4. 「💾 data.json ダウンロード」でファイル保存
5. ダウンロードした data.json をリポジトリのルートに、画像ファイルを images/ フォルダに配置してcommit/push
6. GitHub Pagesに反映後、TikTok Live Studioで overlay.html を読み込む(ブラウザソースサイズはお好みで調整)
