🎁 Gift Overlay Admin

左寄せ版・右寄せ版それぞれを編集 → JSON書き出し → リポジトリにpush

セット一覧

使い方:
左寄せ版と右寄せ版のタブを切り替えて、それぞれ独立して編集できます。
1. タブを選択(左寄せ版 or 右寄せ版)
2. 「＋ セット追加」で新規追加、または「📂 JSON読み込み」で既存ファイルから編集再開
3. 各セットのサムネイル枠に画像をドラッグ&ドロップ(またはクリックで選択)
4. ギフト名・任意テキストを入力、必要なら↑↓で並び替え
5. 「💾 JSON ダウンロード」でファイル保存(ファイル名は自動でタブに対応: data.json または data-right.json)
6. 画像をリポジトリの images/ に、JSONをリポジトリのルートに配置してcommit/push
7. TikTok Live Studioで読み込み:
・左寄せ版: https://yourname.github.io/your-repo/overlay.html
・右寄せ版: https://yourname.github.io/your-repo/overlay.html?align=right
