# Gift Overlay for TikTok Live Studio

TikTok Live Studio用のギフト紹介オーバーレイです。1920×1920の正方形キャンバスに、画像・ギフト名・任意テキストのセットを最大6個ずつ縦に表示し、7個以上ある場合は2秒ごとにページを切り替えてループ表示します。

**左寄せ版**(画像が左、テキストが右に左揃え)と**右寄せ版**(画像が右、テキストが左に右揃え)の2つを同時に運用できます。

## ファイル構成

```
/
├ overlay.html       オーバーレイ本体(URLパラメータで左右切替)
├ admin.html         編集用ページ(タブで左右切替、プレビュー付き)
├ data.json          左寄せ版のセットデータ
├ data-right.json    右寄せ版のセットデータ
├ images/            webp画像置き場
│  ├ rose.webp
│  ├ cat-paw.webp
│  └ baseball.webp
├ fonts/             フォント置き場
│  ├ NotoSansJP-Black.ttf  (オーバーレイで使用する極太フォント)
│  └ NotoSansJP-Bold.ttf   (予備、現状未使用)
└ README.md
```

## セットアップ

1. このリポジトリをGitHubにアップロード
2. リポジトリ設定で **GitHub Pages** を有効化(Settings → Pages → Source: `main` branch)
3. 公開URLを確認(例: `https://yourname.github.io/your-repo/`)

## 使い方

### 編集 (admin.html)

1. ローカルのブラウザで `admin.html` を開く(またはGitHub PagesのURL `/admin.html` を開く)
2. **画面上部のタブ**で「左寄せ版」「右寄せ版」を切り替え。それぞれ独立して編集できる
3. 「📂 JSON読み込み」で既存ファイルから編集再開、または「＋ セット追加」で新規追加
4. 各セットのサムネイル枠に画像ファイルを **ドラッグ&ドロップ**(またはクリックで選択)
   - ファイルが自動でダウンロードされ、画像パスが自動入力される
   - 後でリポジトリの `images/` フォルダに配置する
5. ギフト名・任意テキストを入力。必要なら↑↓ボタンで並び替え
6. 右側のプレビューで実際の見た目を確認(タブと同じ寄せ方向で表示)
7. 「💾 ダウンロード」ボタンを押すと、現在のタブに対応したファイル名で保存される
   - 左寄せ版タブ → `data.json`
   - 右寄せ版タブ → `data-right.json`
8. ダウンロードしたJSONをリポジトリのルートに、画像ファイルを `images/` フォルダに配置
9. `git add`, `git commit`, `git push` でGitHubに反映

### TikTok Live Studio

ブラウザソースを追加し、用途に応じてURLを使い分けます:

- **左寄せ版**: `https://yourname.github.io/your-repo/overlay.html`
- **右寄せ版**: `https://yourname.github.io/your-repo/overlay.html?align=right`

**ブラウザソース設定:**
- 解像度: **幅 1920 × 高さ 1920**
- カスタムCSS: 不要(背景は最初から透明)
- ライブスタジオ側で配置・サイズ調整

両方を同時に表示することもできます(画面の左右に別々のオーバーレイを配置するなど)。

## セットの内容

各セットは以下の3つの項目を持ちます:

```json
{
  "image": "images/rose.webp",
  "name": "ローズ",
  "text": "ありがとう!"
}
```

- **image**: 画像のパス(リポジトリのルートからの相対パス)。168×168の透過webpを推奨
- **name**: ギフト名
- **text**: 任意テキスト
- 両方記入されていれば両方表示(72px)、片方だけ記入されていればその文字を倍サイズ(144px)で表示

## 動作仕様

### レイアウト
- **左寄せ版**: 画像が左、その右側にテキスト。ギフト名と任意テキストの左端が揃う(全セットで縦のライン)
- **右寄せ版**: 画像が右、その左側にテキスト。ギフト名と任意テキストの右端が揃う(全セットで縦のライン)

### 共通
- 1ページに最大6セットを縦に並べて表示(セット間ギャップ14px)
- 7個以上ある場合は2秒ごとに次のページにクロスフェード(0.5秒)で切替
- 最後のページの後は1ページ目に戻ってループ
- セットが6個以下の場合は1ページのみ表示し続ける(切替なし)
- セットが0個の場合は何も表示しない
- 画像は左右に±10°ゆっくり揺れるアニメーション
- テキストは白文字に黒い縁取り、極太のNoto Sans JP Black
- フォント読み込み完了まで全体非表示(チラつき防止)
- JSONを30秒ごとにポーリングするので、push後しばらく待てば自動反映(リロード不要)
